# Compiler

A compiler/parser project developed in C for academic purposes.

## Description

This is an **academic project** that implements a compiler or parser to process and compile source code. The project contains the necessary source files and example files to test functionality. It is designed as an educational tool to understand compiler design and implementation principles.

## Project Structure

```
.
├── main.c                    # Main compiler file
├── emetteur.txt             # Input or configuration file
├── test_pascal.txt              # Example file
├── .gitignore              # Files to ignore by Git
└── README.md               # This documentation
```

## Prerequisites

- **GCC** or any compatible C compiler
- **Make** (optional, to facilitate compilation)

## Installation and Compilation

### Simple Compilation

```bash
gcc main.c -o main
```

### Execution

```bash
./main
```

## Usage

To use the compiler:

```bash
./main [options] [input_file]
```

## Important Files

- **main.c**: Entry point of the compiler
- **emetteur.txt**: Input file for the compiler
- **example.txt**: Usage example

## Academic Purpose

This project is developed for **educational purposes** to:
- Understand the fundamentals of compiler design
- Learn C programming concepts
- Study parsing and code generation techniques
- Develop practical experience with low-level programming

⚠️ This is **NOT** intended for production use.
